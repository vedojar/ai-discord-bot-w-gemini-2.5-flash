# ai-discord-bot-w-gemini-2.5-flash
This bot allows you to chat with an AI robot powered by Gemini 2.5 Flash on your Discord server.
To make the bot work, you need to create a file named .env in the same folder and write: DISCORD_TOKEN=YOUR_DISCORD_BOT_TOKEN
GEMINI_API_KEY=AI_STUDIO_KEY
You need to list the desired attributes for your bot line by line in the BOT_PERSONALITY section.
On line 48, you must enter the ID of the channel where you want to chat with the bot, in the CHAT_CHANNEL_ID field.
